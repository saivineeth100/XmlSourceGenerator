# Getting Started

## Installation

Install via NuGet Package Manager:

```bash
dotnet add package XmlSourceGenerator
```

Or via Package Manager Console:

```powershell
Install-Package XmlSourceGenerator
```

## Your First Model

1. Add the `[XmlAutoGenerated]` attribute to your class
2. Make the class `partial`
3. Add XML attributes to customize serialization

```csharp
using SourceGeneratorUtils;

[XmlAutoGenerated]
public partial class Person
{
    public int Id { get; set; }
    public string Name { get; set; }
    public DateTime BirthDate { get; set; }
}
```

## Serialization

The generator creates `WriteToXml()` and `ReadFromXml()` methods:

```csharp
// Serialize to XML
var person = new Person 
{ 
    Id = 1, 
    Name = "Alice", 
    BirthDate = new DateTime(1990, 5, 15) 
};

XElement xml = person.WriteToXml();
Console.WriteLine(xml);

// Output:
// <Person>
//   <Id>1</Id>
//   <Name>Alice</Name>
//   <BirthDate>1990-05-15T00:00:00</BirthDate>
// </Person>
```

## Deserialization

```csharp
// Deserialize from XML
var xml = XElement.Parse(@"
<Person>
  <Id>1</Id>
  <Name>Alice</Name>
  <BirthDate>1990-05-15T00:00:00</BirthDate>
</Person>");

var person = new Person();
person.ReadFromXml(xml);

Console.WriteLine($"{person.Name} - {person.BirthDate:yyyy-MM-dd}");
// Output: Alice - 1990-05-15
```

## Runtime Options

Customize serialization at runtime with `XmlSerializationOptions`:

```csharp
var options = new XmlSerializationOptions
{
    PropertyNamingPolicy = XmlNamingPolicy.CamelCase,
    WriteIndented = true
};

var xml = person.WriteToXml(options);
```
